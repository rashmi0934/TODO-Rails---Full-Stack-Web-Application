# TodoRails

A polished, secure, and fully functional task management web application built with Java and Spring Boot. This project was developed as a portfolio-ready application that demonstrates practical full-stack development skills, including authentication, database integration, MVC architecture, and user-friendly UI design.

## 🚀 Overview

TodoRails is a web-based productivity app that allows users to register, log in, create tasks, manage deadlines, mark tasks as complete, and update their account securely. The application combines a clean front-end experience with a robust backend powered by Spring Boot.

This project reflects strong fundamentals in:
- Java backend development
- Spring Framework and Spring Security
- Database-driven web applications
- MVC-based application structure
- Clean, user-focused UI implementation

---

## ✨ Features

- Secure user registration and login
- Password change and profile management
- Create, view, edit, complete, and delete tasks
- Dashboard with task overview and due-date organization
- Responsive and attractive interface
- Data persistence using MySQL and JPA

## 📁 Project Structure

- Controllers: Manage routing and application flow
- Services: Handle business logic for users and tasks
- Models: Define the core entities
- Repositories: Interact with the database
- Templates: Render the web pages
- Resources: Contain CSS, JavaScript, and configuration files

## ▶️ Getting Started

### Prerequisites
- Java 21
- Maven
- MySQL Server

### Run Locally
1. Create a MySQL database named `todorails`.
2. Update your database credentials in `src/main/resources/application.properties`.
3. Start the application:

```bash
./mvnw spring-boot:run
```

4. Open the app in your browser:

```text
http://localhost:8080
```
